# Habit Tracker

A simple, offline, open-source habit tracker.

No accounts.  
No subscriptions.  
No data collection.  

Just track your habits and improve your life.

---

## Why I built this

I struggled with tracking my habits.

I tried using notebooks, but I would forget or stop after a few days.

Then I discovered people using Google Sheets habit trackers with charts.

I loved the idea.

But every good template I found was paid, and learning Google Sheets formulas felt complicated.

So I thought:

> I'm a programmer. Why don't I just build my own?

I built this tool for myself first.

Then a friend started using it and really liked it.

So I decided to make it **open source and free for everyone.**

---

## Features

- Track daily habits
- Simple check system
- Progress visualization
- Works offline
- Installable on your phone
- No login required
- No data tracking
